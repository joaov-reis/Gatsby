# Objetivo

Este projeto utiliza do framework Gastby e conceitos de Jamstack para a criação da página de uma empresa de alimentos matinais, com integração de APIs externas, preparação para deploy no CDN Netlify, importação de informações internas através do GraphQL e estilização básica.

## Dependências para este projeto
As seguintes versões foram utilizadas neste projeto:

- node@22.20.0
- npm@11.6.2
- react@18.2.0
- gatsby@5.14.6

## Instalação inicial do projeto
1. Os arquivos package.json e tsconfig.json já estão pré configurados, instale as dependências necessárias com o comando abaixo no terminal:

```bash
npm i
```

## Configurações

### Debugger

1. Rode o script de build para criar os arquivos de produção:

```bash
npm run develop
```

## Executando o código

1. Rode o script de dev para executar o servidor na porta http://localhost:8000

```bash
npm run develop
```
